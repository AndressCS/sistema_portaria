# Sistema de Portaria

Sistema desenvolvido em Python para controle de entrada e saída de motoristas.

## Funcionalidades

- Cadastro de motoristas
- Registro de entrada e saída
- Controle de acesso
- Histórico de movimentações

## Tecnologias

- Python
- SQLite
- Tkinter (se estiver usando)
